Bsc(Hons) Software Engineering
Duration :
4-Years
Course Type :
Full Time

Year 1:

The stage 1 of the programme establish a strong frameworks in computing, in order to provide fundamental background knowledge. The course also features a range of non-technical modules including legalization and ethical considerations to enhance students job skills.

  • Introduction to Computer Science
  • Mathematics for Computing 1
  • Programming with C Language
  • Database Management Systems
  • Internet Technology
  • Data Structures & Algorithms
  • Computer Technology
  • Object Oriented Programming with C#
  • System Analysis and Design
  • ntroduction to Computer Networks

Year 2:

The second develops your understanding in your specialist subject area, Computer Networks, Computer Security and Software Engineering and link it with other relevant areas within Computing. This will cover all the core content and other specialist area topics as well as all critical issues of any IT infrastructure.
  • Human Computer Interaction
  • Databases
  • Software Development Platform
  • IT Legislation & Ethics
  • Network & System Administration
  • Network Security
  • Integrating Project
  • Advanced Database Management Systems

Year 3:

Final year provides the required enhanced understanding and mastering the latest technologies in your chosen specialist area of study, with a string focus on integrating these skills with current business needs and industry demands. Critical part of the final year is the project, which challenges and enhances design, development, and evaluation skills. This allows graduates to demonstrate the ability to undertake research and development activities within commercial environments.
  • Software Development and Project Mgt
  • Cross-Platform Development in C++
  • Incident Prevention, Detection & Response
  • Network Monitoring
  • Project
  • API Software Development
  • Advanced Databases and Their Applications
  • Distributed Systems
